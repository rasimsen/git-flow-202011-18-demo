# Zero To Hero - Demo by using Git Flow

## Description

- Github is used as a repo management tool
- Git and Git Flow initialised in this project
- NestJS is used to create Node.js application

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
